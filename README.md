# Asteroids

* play the classic retro game
* large & medium asteroids split into smaller ones when shot
* player can rotate, thrust forwards & backwards, and shoot

## Installation & Usage

* `uv venv`
* `source .venv/bin/activate`
* `uv sync`
* `uv run main.py`
